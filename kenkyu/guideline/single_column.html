シングルカラム

single column
出展： https://creators.uuum.jp/

シングルカラムのような「コンテンツ中心のデザイン」では、コンテンツを伝えるだけではなく、ユーザーインタフェースにも気を配る必要がある。

何故ならば、WEBサイトの見やすさや使いやすさといった要素は、そのままそのサイトの「親しみ」にへと直結する。つまり、コンテンツ中心であっても、ユーザーインタフェースによってその印象は変わってしまうからである。

ディスプレイを最大限に使うことのできるシングルカラムでは、全体的にシンプルな構成になりやすい。そのため、どこがクリック出来るかといった「クリック要素の区別」が必要となってくる。


クリック要素の判別

ホバーエフェクト

BUTTON コードを見る 
         .single_hover{
           background: #607D8B;
           color: #FFFFFF;
           text-decoration: none;
           font-size: 1em;
           letter-spacing: 0.05em;/*字間*/
           padding: 0.2em 1em;/*ボタン内の余白*/
           border-radius: 3px;/*角の丸み*/
           transition: .5s;
         }
         .single_hover:hover{
          background: #FFFFFF;
          color: #607D8B;
          border: solid 0.5px #607D8B;
         }

シャドウ

BUTTON コードを見る 
         .single_shadow{
           background: #607D8B;
           color: #FFFFFF;
           text-decoration: none;
           font-size: 1em;
           letter-spacing: 0.05em;/*字間*/
           padding: 0.2em 1em;/*ボタン内の余白*/
           border-radius: 3px;/*角の丸み*/
           transition: .5s;
           box-shadow: 0 2px 2px 0 rgba(0,0,0,0.14), 0 1px 7px 0 rgba(0,0,0,0.12), 0 3px 1px -1px rgba(0,0,0,0.2);/*浮き上がるように*/
         }
         .single_shadow:hover{
           box-shadow: 0 4px 10px 0 rgba(0,0,0,0.14), 0 1px 5px 0 rgba(0,0,0,0.12), 0 3px 1px -2px rgba(0,0,0,0.2);/*影*/
         }

アクセントカラー

BUTTON コードを見る 
         .single_accent{
           background: #FDD835;
           color: #000000;
           text-decoration: none;
           font-size: 1em;
           letter-spacing: 0.05em;/*字間*/
           padding: 0.2em 1em;/*ボタン内の余白*/
           border-radius: 3px;/*角の丸み*/
           box-shadow: 0 2px 2px 0 rgba(0,0,0,0.14), 0 1px 7px 0 rgba(0,0,0,0.12), 0 3px 1px -1px rgba(0,0,0,0.2);/*浮き上がるように*/
         }
         .single_accent:hover{
           box-shadow: 0 4px 10px 0 rgba(0,0,0,0.14), 0 1px 5px 0 rgba(0,0,0,0.12), 0 3px 1px -2px rgba(0,0,0,0.2);/*影*/
         }

クリック要素の判別をデザインするには、ホバーエフェクト、シャドウ、アクセントカラーといった手法が効果的である。

特にホバーエフェクトとシャドウは視覚的にも触覚的にもわかりやすく、WEBを扱うのに慣れていない人でも操作を理解させやすい装飾と言える。

加えて、色をあまり使わないデザインにおいても反映させやすいため、あらゆるデザインに汎用的に使うことができる。

ただしアクセントカラーに関しては、その色が使われた要素によってデザイン性が下がってしまう場合があるため、使用する際にはよく検討する必要がある。
